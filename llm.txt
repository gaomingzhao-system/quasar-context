# Quasar UI

> A high-performance Vue-based framework and UI library for Vue 3 that lets you build SPA, SSR, PWA, mobile, desktop, and browser extensions from a single codebase.

## Composables

Quasar exposes Vue composables for accessing framework features in the Composition API.

- [useQuasar](https://quasar.dev/vue-composables/use-quasar/): Access the `$q` object (platform, screen, language, dark mode, plugins, etc.) inside `setup()`. :contentReference[oaicite:76]{index=76}  
- [useMeta](https://quasar.dev/vue-composables/use-meta/): Composition API interface to the Meta plugin for dynamic `<title>` and `<meta>` tags. :contentReference[oaicite:77]{index=77}  
- [useId](https://quasar.dev/vue-composables/use-id/): Generate SSR-safe, reactive unique IDs to bind to DOM attributes. :contentReference[oaicite:78]{index=78}  
- [useSplitAttrs](https://quasar.dev/vue-composables/use-split-attrs/): Split `attrs` into listener vs. regular HTML attributes for advanced component design. :contentReference[oaicite:79]{index=79}  

Additional composables are documented under the [Vue Composables](https://quasar.dev/vue-composables/) section.

## Plugins & Utils

Quasar plugins integrate cross-cutting concerns like notifications, dark mode, and storage.

- [Notify](https://quasar.dev/quasar-plugins/notify/): Floating toast/snackbar notifications, often used with `$q.notify()` from `useQuasar`. :contentReference[oaicite:80]{index=80}  
- [Loading](https://quasar.dev/quasar-plugins/loading/): Global loading overlay with spinner and message for background operations. :contentReference[oaicite:81]{index=81}  
- [Dark](https://quasar.dev/quasar-plugins/dark/): Centralized dark mode state management with configurable modes (`on`, `off`, `auto`). :contentReference[oaicite:82]{index=82}  
- [Meta Plugin](https://quasar.dev/quasar-plugins/meta/): Manage SEO meta tags, head scripts/styles, and `<html>/<body>` attributes. :contentReference[oaicite:83]{index=83}  
- [App Visibility](https://quasar.dev/quasar-plugins/app-visibility/): React to whether the app is visible or in focus, via Page Visibility API. :contentReference[oaicite:84]{index=84}  
- [Addressbar Color](https://quasar.dev/quasar-plugins/addressbar-color/): Control browser address bar color on mobile. :contentReference[oaicite:85]{index=85}  
- [Cookies](https://quasar.dev/quasar-plugins/cookies/): Simple API for managing cookies. :contentReference[oaicite:86]{index=86}  
- [Local/Session Storage](https://quasar.dev/quasar-plugins/web-storage/): Helpers for `localStorage` and `sessionStorage`. :contentReference[oaicite:87]{index=87}  

Utility modules are also provided:

- [Date Utils](https://quasar.dev/quasar-utils/date-utils/): Helpers to manipulate and format dates. :contentReference[oaicite:88]{index=88}  
- [DOM Utils](https://quasar.dev/quasar-utils/dom-utils/): DOM helpers such as position, selection, and element helpers. :contentReference[oaicite:89]{index=89}  
- [Formatter Utils](https://quasar.dev/quasar-utils/formatting-utils/): Utilities for string and number formatting. :contentReference[oaicite:90]{index=90}  
- [Scrolling Utils](https://quasar.dev/quasar-utils/scrolling-utils/): Programmatic scrolling helpers for pages and elements. :contentReference[oaicite:91]{index=91}  
- [Type Checking Utils](https://quasar.dev/quasar-utils/type-checking-utils/): Functions for detecting variable types. :contentReference[oaicite:92]{index=92}  
- [Event Bus Util](https://quasar.dev/quasar-utils/event-bus-util/): Lightweight global event bus helper. :contentReference[oaicite:93]{index=93}
